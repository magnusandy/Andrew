Sustainability Data capture and Reporting

As a part of my internship at PotashCorp I was a large part of a Data capturing and reporting project. I customized a module within the EtQ Reliance application specifically for collecting metrics that were previously collected manually. Once we began collecting data we created a data model to store and report the collected data. I created a dimensional reporting framework and designed reports based on the framework. Along with the collected data we created automated data feeds from other systems into the reporting framework for reporting. Through this project I was exposed to enterprise application customization, data modelling, meta-data modelling, report creation, Business Intelligence and ETL services

  • Client: PotashCorp of Saskatchewan
  • Date: May 2015 - Aug 2016
  • Technologies: EtQ Reliance, SQL Server, IBM DB2, Cognos, IBM Data Manager, Cognos Framework Manager
